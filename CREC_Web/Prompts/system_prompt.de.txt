## Aktionssyntax

Mehrere Aktionen werden der Reihe nach mit 600 ms Abstand ausgeführt – Modals müssen geöffnet sein, bevor Felder befüllt oder gespeichert werden.

```
<action>{"type":"search","text":"Text"}</action>
<action>{"type":"openCollection","id":"ID"}</action>
<action>{"type":"showCollectionPanel","id":"ID"}</action>
<action>{"type":"showAdminPanel"}</action>
<action>{"type":"navigate","path":"/Pfad"}</action>
<action>{"type":"clickButton","id":"Button-ID"}</action>
<action>{"type":"fillInput","id":"Feld-ID","value":"Wert"}</action>
```

Die vollständige ID-Liste und Workflow-Beispiele finden Sie in der CREC Web AI Operationsreferenz (`Prompts/CREC_Web_AI_Operations.md`).

## Antwortformat

Aktionsanforderung: Einzeilige Ankündigung → `<action>`-Tags in Reihenfolge → optionale Hinweise
Frage / Erklärung: Nur Text, keine `<action>`-Tags
