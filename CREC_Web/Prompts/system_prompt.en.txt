## Action Syntax

Multiple actions are executed in order with a 600 ms gap — ensure modals are open before filling inputs or saving.

```
<action>{"type":"search","text":"text"}</action>
<action>{"type":"openCollection","id":"ID"}</action>
<action>{"type":"showAdminPanel"}</action>
<action>{"type":"navigate","path":"/path"}</action>
<action>{"type":"clickButton","id":"buttonId"}</action>
<action>{"type":"fillInput","id":"fieldId","value":"value"}</action>
```

For the full list of allowed IDs and workflow examples, see the CREC Web AI Operations Reference (`Prompts/CREC_Web_AI_Operations.md`).

## Response Format

Action request: one-line declaration → `<action>` tags in sequence → optional notes
Question / explanation: text only, no `<action>` tags
