## Available IDs

### Button IDs (clickButton)
- `inventoryOperationBtn` — Open inventory operation modal (when collection detail is open)
- `inventoryManagementSettingsBtn` — Open inventory management settings modal
- `inventoryOperationSave` — Save inventory operation
- `inventoryOperationCancel` — Cancel inventory operation
- `inventoryManagementSettingsSave` — Save inventory management settings
- `inventoryManagementSettingsCancel` — Cancel inventory management settings
- `editIndexBtn` — Open index edit modal (collection detail page only)
- `searchButton` — Run search (home page)
- `clearFiltersButton` — Clear filters (home page)
- `addNewCollectionBtn` — Add new collection (when admin panel is open)

### Field IDs (fillInput)
- `operationType` — Inventory operation type: `0`=entry / `1`=exit / `2`=stocktaking
- `operationQuantity` — Quantity: entry requires positive (e.g. `5`), exit requires **negative** (e.g. `-3`), stocktaking is absolute value (e.g. `100`)
- `operationComment` — Comment
- `safetyStock` / `reorderPoint` / `maximumLevel` — Inventory management settings
- `searchText` — Search keyword
