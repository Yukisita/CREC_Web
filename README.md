# CREC Web

**CREC Web** は、物理的なコレクションをデジタルで一元管理するための Web ベースのコレクション管理システムです。

## 概要

博物館の収蔵品・図書館の蔵書・工業製品・在庫品など、さまざまな物理アイテムをデジタル化して管理できます。コレクションに対してメタデータ・画像・動画・各種ファイルを紐付け、ブラウザから直感的に検索・閲覧・編集が可能です。

## 主な機能

- **コレクション管理** — 名称・管理コード・カテゴリ・タグ・場所などのメタデータを登録・編集・削除
- **マルチメディア対応** — 画像・動画・各種ファイルをコレクションに添付して管理
- **高度な検索** — 全文検索・フィールド指定・検索方式（部分一致 / 前方一致 / 後方一致 / 完全一致）・在庫状況によるフィルタリング
- **在庫管理** — 入庫・出庫・棚卸し操作の記録、適正在庫・発注点・最大在庫の設定
- **QR コードスキャン** — カメラを使ったコレクションの即時検索（HTTPS 環境が必要）
- **多言語対応** — 日本語・英語の切り替えに対応
- **カスタムラベル** — 各フィールドの表示名をプロジェクトごとにカスタマイズ可能

## 技術スタック

| 項目 | 内容 |
|------|------|
| フレームワーク | ASP.NET Core MVC (.NET 8.0) |
| フロントエンド | Bootstrap + Vanilla JavaScript |
| データ保存 | ファイルシステムベース（JSON + バイナリファイル） |
| 言語 | C# |

## データ構造

コレクションデータはファイルシステム上に以下の構造で保存されます。

```
{データフォルダ}/
├── {コレクションID}/
│   ├── SystemData/
│   │   ├── index.json        # メタデータ
│   │   └── inventory.json    # 在庫管理データ
│   ├── pictures/             # 画像ファイル
│   ├── videos/               # 動画ファイル
│   └── data/                 # その他のファイル
└── $SystemData/
    └── RecycleBin/           # 削除済みコレクション
```
